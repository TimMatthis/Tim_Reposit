How it works

  1. Average Daily Profile — year of 5-min readings is folded into a single representative 24-hour curve per home.
  2. Min-Max Scaling — each profile is normalised to [0–1] so clustering captures shape, not volume.
  3. KShape Clustering — uses normalised cross-correlation to group homes by behavioural pattern (e.g. Evening Peakers, Solar Exporters, Flatliners).
  4. Results — a visualisation and a CSV mapping each home to its cluster.
