Product Configuration

How to read scores: Each level’s part-worth is whatever the data says (imported win rates or the defaults below). Higher part-worth = more preferred for that level. Default installation is $0 (80) > $500 (44.4) — so the optimizer and “maximize” pick $0 unless you 🔒 lock a different rung. A strong “signal strength” for installation only means that attribute moves the score a lot between levels, not that expensive levels are better.

Simulated prices: Check “Interpolate…” under a dollar attribute to set any price between (or beyond) conjoint levels. Part-worths follow straight segments between tested points (Sawtooth-style linear interpolation); outside the tested range the nearest endpoint is used. With interpolation enabled, “Find Top 3” searches a price grid for those attributes (see note under that section).
